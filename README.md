# 🔐 Password Generator
A sleek, secure, and responsive web application designed to generate strong, customizable passwords to enhance digital security.

## 🚀 Live Demo
You can try the live application here: 
**[View Project Live](https://kulshrestha-sanket123.github.io/Password-Generator/)**

---

## ✨ Key Features
* **Customizable Length:** Generate passwords from 6 to 12 characters.
* **Complexity Control:** Options to include Uppercase, Lowercase, Numbers, and Special Symbols.
* **Strength Indicator:** Real-time feedback on password complexity.
* **One-Click Copy:** Integrated clipboard functionality for ease of use.
* **Mobile Responsive:** Fully optimized for all screen sizes (Mobile, Tablet, Desktop).

---

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Modern UI/UX)
* **Logic:** Vanilla JavaScript (ES6+)
* **Deployment:** GitHub Pages

---

## 🛡️ Security & Best Practices
* **Zero Hardcoding:** Followed industry standards by ensuring no sensitive API keys or secrets are stored in the frontend.
* **Client-Side Logic:** Passwords are generated locally on the browser, ensuring no data is sent to a server.
